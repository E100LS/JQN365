# 我的炒股经验总结 - 个人博客网站

## 项目简介

一个基于 React + TypeScript 开发的个人博客网站，主题为【我的炒股经验总结】。

## 功能特性

- ✅ **文章列表** - 展示所有博客文章，支持标签筛选
- ✅ **文章详情** - Markdown 渲染，支持代码高亮、表格、引用等
- ✅ **标签分类** - 按标签筛选文章（技术分析、基本面分析、心态修炼等）
- ✅ **关于页面** - 作者介绍和博客说明
- ✅ **响应式设计** - 适配手机、平板、桌面端
- ✅ **前后导航** - 在文章详情页可切换上一/下一篇

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| React | 19 | 前端框架 |
| TypeScript | 6.0 | 类型安全 |
| Vite | 8.1 | 构建工具 |
| Tailwind CSS | 3.4 | 样式框架 |
| React Router DOM | 7 | 路由管理 |
| React Markdown | 10.1 | Markdown 渲染 |

## 快速开始

```bash
cd stock-blog
npm install
npm run dev
```

开发服务器地址：http://localhost:5173
